# AutoFácil - Simulador de Cuotas

Aplicación web para calcular cuotas de financiamiento automotriz.

## Características

- Cálculo de cuotas para 12, 24, 36 y 48 meses
- Incluye Seguro de Desgravamen, RDH y Reparaciones Menores
- Tasa variable según monto a financiar
- Optimizado para dispositivos móviles
- Instalable como PWA (Progressive Web App)

## Despliegue en Vercel

1. Sube este proyecto a GitHub
2. Conecta tu repositorio con Vercel
3. Vercel detectará automáticamente la configuración
4. ¡Listo! Tu app estará en línea

## Desarrollo Local

```bash
npm install
npm run dev
```

## Build para Producción

```bash
npm run build
```
